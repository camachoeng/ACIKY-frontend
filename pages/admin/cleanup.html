Limpieza Automática

La limpieza se ejecuta automáticamente cada noche a las 3:00 AM UTC comparando Cloudinary con la base de datos de producción.

Para activar: configurar Heroku Scheduler con el comando node scripts/cleanup.js cada 24 horas.
